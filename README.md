# MindMap
Basic HTML MindMap

Double-click on page to create a new node.

Click on node to select and then on any other node to create a path. 

Double-click on node to modify its label.

Right-click on node to show/hide its children.

Use the Theme dropdown menu to color the new node.

After creating your personal map, don't forget to export it as json for later!
